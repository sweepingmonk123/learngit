NOTE: 本文档适用于如下版本：

每个版本的具体功能请参考<<RD-OS版本信息>>

[cols="1,3", options="header"]
|===
|版本
|更新内容
|{V100image}
|RD-OS的基础版本，拥有数据集成、任务开发、运维中心等基础功能，此后的所有功能都是在这个版本的基础上叠加的
|{V110image} +
a|
* 新增数据管理功能，可查看HDFS内的所有表信息，包括基础信息（表名、存储空间等）、字段信息、分区信息，同时可预览数据；
* 页面多处功能优化，修改部分bug；
|{V120image}
a|
* 进行了多处底层优化，包括spark和flink的yarn提交、flink V1.3版本支持等；
* 优化HDFS数据集成任务模块，可快速配置大量字段信息；
* 新增HBase数据集成模块，可对HBase存储单元进行读写；
* 新增实时计算kafka数据延迟统计功能，可实时查看数据在kafka内的延迟情况，并根据配置触发告警；
|===
