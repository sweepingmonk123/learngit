== 数栈·数据中台产品体系介绍

本文档只是DTinsightIDE（开发套件）的产品使用手册，但在详细介绍开发套件产品之前，可以先了解一下数栈·数据中台整体的产品体系：

image::images\01_DTinsightIDEHelp_Summary-4ecad.png[]

* **EasyV**：秒级可视化大屏开发平台，数据价值触手可及。
* **数据API（API）**：可视化配置数据API，统一监控与审批，实现API的一站式配置、审批与监控。
//详情请参考<<DTinsightAPI>>
* **数据开发套件（IDE）**：一站式大数据开发平台，帮助企业快速完全数据中台搭建。
* **数据质量（Valid）**：对过程数据和结果数据进行质量校验，帮助企业及时发现数据质量问题。
//详情请参考<<DTinsightValid>>
* **数据地图（Map）**：可视化的数据资产中心，帮助企业全盘掌控数据资产情况和数据的来源去向。
//详情请参考<<DTinsightValid>>
* **数据模型（Model）**： 使企业数据标准化，模型化，帮助企业实现数据管理规范化。
//详情请参考<<DTinsightModel>>
* **玳数日志（EasyLog）**：海量日志管理、分析，快速生成监控告警场景，提升故障发现率、降低运维成本。
//详情请参考<<DTEasyLog>>

== 数据质量产品介绍

DTinsightValid（数据质量）是由
http://www.dtstack.com[袋鼠云]开发的一款数据质量稽核工具，支持对过程数据和结果数据进行校验，帮助企业及时发现数据质量问题，本文档是DTinsightValid（数据质量）的使用手册。

NOTE: 本文档适用于如下版本：
 {V230image}

袋鼠云DTinsightValid，是一款高效的数据质量稽核工具，支持对接MySQL、Oracle、Hive等不同类型的存储单元，支持对数据表进行表级、字段级数据质量稽核，内置20余项校验规则。帮助企业及时发现数据质量问题，以产品化的形式动态监控数据质量。

* **主要特性**

* 丰富的数据源支持

支持MySQL、Oracle、SQLServer、MaxCompute、Hive、PostgreSQL数据源，可对这些数据源内的数据表进行多维校验。

* 内置丰富的稽核规则

内置20余种校验规则，满足绝大多数的数据稽核场景。
